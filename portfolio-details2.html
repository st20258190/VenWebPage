Restaurant Website for outer clove

Project information

  • Category: Php Web Application
  • Client: Outer Clove Restaurant
  • Project date: 15 November, 2023

As part of my university project, I developed a dynamic web application for the Outer Clove Restaurant using HTML, CSS, JavaScript, and PHP. This application provides a comprehensive platform for the restaurant, featuring an attractive and responsive design. It includes essential functionalities such as an interactive menu, online reservations, contact information, and customer reviews. Additionally, I implemented an admin panel that allows the restaurant staff to easily manage menu items, reservations, and other content. The project showcases my ability to create functional and visually appealing web solutions that enhance the user experience and operational efficiency of the restaurant.
